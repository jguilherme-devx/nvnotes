# NVNotes

![Demonstração](/assets/images/example.jpg)

  NVNotes is a note manager, created for Termux and written in Python.

## Features

- Create notes
- View notes
- Edit notes
- Remove notes
- Markdown support
- Rich terminal interface

## Requirements

 ● Termux

 ● Nvim

 ● git

 ● Pyhton

## Installation

```bash
pkg install python git -y
git clone https://github.com/jguilherme-devx/nvnotes.git
cd nvnotes
python scripts/install.py
```

## Running

```bash
python NVNotes.py
```
